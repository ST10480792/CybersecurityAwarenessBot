# Cybersecurity Awareness Bot - Part 1
A C# Console application designed to educate South African citizens on cyber threats.

## Setup Instructions
1. Clone the repository.
2. Ensure .NET 6.0 SDK (or higher) is installed.
3. Open the `.sln` file in Visual Studio 2022.
4. Build and Run (F5).

## Features
- Voice Greeting integration
- ASCII Art Logo
- Personalised User Interaction
- Defensive Input Validationm
